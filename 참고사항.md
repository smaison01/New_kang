# 참고사항

```

├── backend/                        ← Spring Boot 백엔드
│   ├── .mvn/wrapper/
│   │   └── maven-wrapper.properties
│   ├── mvnw.cmd                    ← Maven Wrapper (mvn 대신 사용)
│   ├── pom.xml
│   └── src/main/
│       ├── java/com/example/board/
│       │   ├── BoardApplication.java
│       │   ├── controller/PostController.java
│       │   ├── entity/Post.java
│       │   ├── repository/PostRepository.java
│       │   └── service/PostService.java
│       └── resources/
│           └── application.properties
├── frontend/                       ← React + Vite 프론트엔드
│   ├── src/
│   │   ├── App.jsx                 ← 메인 게시판 컴포넌트
│   │   ├── App.css                 ← 스타일
│   │   └── api.js                  ← axios API 모듈
│   └── package.json
├── start-backend.cmd               ← 백엔드 실행 스크립트
├── start-frontend.cmd              ← 프론트엔드 실행 스크립트
├── worklog.md
└── 참고사항.md
```

## 실행 방법

### 1. 백엔드 실행
터미널 1에서:
```
start-backend.cmd
```
또는 직접:
```
cd backend
mvnw.cmd spring-boot:run
```
- 최초 실행 시 Maven (~50MB) 자동 다운로드됨 (시간 소요)
- 서버 주소: http://localhost:8080

### 2. 프론트엔드 실행
터미널 2에서:
```
start-frontend.cmd
```
또는 직접:
```
cd frontend
npm run dev
```
- 주소: http://localhost:5173

## API 엔드포인트

| 메서드 | URL              | 설명         |
|--------|------------------|--------------|
| GET    | /api/posts       | 전체 목록    |
| GET    | /api/posts/{id}  | 단건 조회    |
| POST   | /api/posts       | 게시글 작성  |
| PUT    | /api/posts/{id}  | 게시글 수정  |
| DELETE | /api/posts/{id}  | 게시글 삭제  |

## 주의사항

- **H2 인메모리 DB 사용 중** — 백엔드 재시작 시 데이터가 사라집니다.
  실제 DB(MySQL, PostgreSQL 등)를 연결하려면 `application.properties`와 `pom.xml` 의존성을 수정해야 합니다.
- CORS는 `http://localhost:5173`만 허용 중. 배포 시 origin 수정 필요.
- 백엔드를 먼저 실행한 후 프론트엔드를 실행해야 API 요청이 정상 동작합니다.
