와이어프레임 설명

  • ① 신규 등록 폼 : 시작일~종료일(기본값 오늘) + 제목 입력 후 [등록] 클릭 → 목록 갱신
  • ② 조회 기간 필터 : 시작일~종료일 범위로 검색, [초기화] 시 전체 복원
  • ③ 일괄삭제 바 : 체크된 건수 표시, [선택 삭제] 클릭 → confirm 후 일괄 제거
  • ④ 체크박스 : 헤더 체크박스로 전체 선택/해제, 체크된 행은 붉은 배경으로 표시
  • 제목/메모 통합 : 기존 제목+메모 두 컬럼을 제목 단일 컬럼으로 통합
  • 인라인 편집 : [수정] 클릭 시 해당 행 제목 인풋 활성화, [저장]/[취소] 버튼 표시
  • 개별 삭제 : 행의 [삭제] 버튼 → confirm 후 단건 제거
