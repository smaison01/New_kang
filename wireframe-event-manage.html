💡 와이어프레임 설명
  • 신규 등록 : 시작일 → 종료일(기본값 오늘) + 내용 입력 → [+ 등록] 클릭
  • 조회 기간 : 시작일 → 종료일 범위 검색, [초기화] 시 전체 복원
  • 날짜 정렬 : 기본 날짜 내림차순(최신순) 조회 — 헤더 "날짜 ↓" 로 표시
  • 컬럼 : 체크박스 · # · 날짜 · 내용(제목+메모 통합) · 관리
  • 멀티 체크박스 : 헤더 체크로 전체 선택/해제, 선택 행은 연분홍 배경
  • 일괄 삭제 : 선택 건수 + [선택 삭제] → confirm 후 일괄 제거
  • 인라인 편집 : [수정] 클릭 → 해당 행 내용 인풋 활성화(노란 배경)
  • 디자인 : Notion 스타일 — 흰 배경, 회색 사이드바, 미니멀 보더, Inter 폰트
